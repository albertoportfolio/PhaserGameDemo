# Star Collector Demo Game

A simple test game built with **React** and **Vite**. Players move a character to collect stars while avoiding bombs. Touch a bomb and the game ends!

## Features

- Collect stars to increase your score
- Avoid bombs or you lose
- Built with React for interactive UI
- Fast development with Vite
